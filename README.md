# it-meme-claude-spinverbs

Русские IT-мемы для спиннера Claude Code. 63 фразы из IT-фольклора, олбанского языка и геймерской культуры.

Вместо стандартного "Thinking..." ваш Claude будет показывать:

```
⠋ Патчу KDE2 под FreeBSD...
⠙ Деплою в пятницу...
⠹ ЛИИИРОЙ ДЖЕНКИНС...
⠸ Костылю...
⠼ Семь бед — один ресет...
```

## Быстрый старт

```bash
npx it-meme-claude-spinverbs install
```

Готово! Перезапустите Claude Code.

## Команды

| Команда | Описание |
|---------|----------|
| `install` | Заменить стандартный спиннер на IT-мемы |
| `append` | Добавить IT-мемы к стандартным фразам |
| `uninstall` | Вернуть стандартный спиннер |
| `list` | Показать все 63 фразы по категориям |
| `preview` | Показать 5 случайных фраз |

## Добавить свои фразы

Фразы хранятся в `~/.claude/settings.json`:

```json
{
  "spinnerVerbs": {
    "mode": "replace",
    "verbs": ["Ваша фраза 1", "Ваша фраза 2"]
  }
}
```

- `"mode": "replace"` — полностью заменить стандартные фразы
- `"mode": "append"` — добавить к стандартным
